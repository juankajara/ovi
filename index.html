Diseños de Sitios Web - Creación de un OVI en formato web

Objetivo general:

El objetivo de este ovi es dar una explicación acerca de los distintos tipos de ordenamiento que existen en la programación orientada a objetos.

Objetivo específico 1:

Dar una explicación inicial acerca del paradigma actual en la programación orientada a objetos.

Objetivo específico 2:

Explicar de forma inicial el tipo de ordenamiento que se hará por medio de texto para comprender de forma teórica el método a utilizar.

Objetivo específico 3:

Incluir un archivo multimedia de la plataforma youtube con un paso a paso de cómo realizar a través de un lenguaje de programación el ordenamiento escogido.
